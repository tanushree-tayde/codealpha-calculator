# codealpha-calculator
CodeAlpha Internship Task: Basic Calculator using HTML, CSS, and JavaScript. 

## 🔗 Live Demo
https://tanushree-tayde.github.io/codealpha-calculator/

## 🛠️ Built With
* HTML5 & CSS3 (Flexbox/Grid layout)
* JavaScript (ES6 logic for computations and keyboard support)

## 🚀 Key Features
* Core math operations (addition, subtraction, multiplication, division, and percentage)
* Preventing double operators (e.g., stopping users from typing `++`)
* Complete physical keyboard support (Numbers, Backspace, Escape, and Enter keys)
* Automated floating-point decimal cleanup using `.toFixed()`

## 💻 How to Run Locally
1. Download or clone this repository.
2. Open the `index.html` file in any modern web browser.
